# Functions

## `add`

This function adds two summands.

### Arguments

- `a`: `number` - The first summand.
- `b`: `number` - The second summand.

### Returns

`number`

The sum.

## `subtract`

This function subtracts a subtrahend from a minuend.

### Arguments

- `a`: `number` - The minuend.
- `b`: `number` - The subtrahend.

### Returns

`number`

The difference.
